## Техническое задание (ТЗ) для стажера PHP программиста: Разработка простого веб-приложения "TaskManager"

**Цель проекта:**
Создать базовое веб-приложение "TaskManager" для управления задачами. Приложение должно позволять пользователям добавлять, редактировать, удалять и просматривать задачи. Это задание направлено на ознакомление стажера с основами PHP и баз данных.

**Требования к функционалу:**

1. **Авторизация и Регистрация:**
   - Реализовать страницу входа (логина) для пользователей.
   - Предоставить возможность регистрации новых пользователей с использованием электронной почты и пароля.

2. **Управление задачами:**
   - Авторизованные пользователи могут создавать новые задачи, указывая заголовок, описание и срок выполнения.
   - Пользователи могут просматривать список своих задач, редактировать существующие и удалять задачи.

3. **Страница с деталями задачи:**
   - При клике на задачу, открывать страницу с подробной информацией о задаче (заголовок, описание, срок выполнения).

4. **Фильтрация и Сортировка:**
   - Предоставить возможность фильтрации задач по статусу (выполнено/не выполнено) и дате завершения.
   - Реализовать возможность сортировки задач по сроку выполнения (по возрастанию/убыванию).

5. **Безопасность и Проверка данных:**
   - Обеспечить безопасность данных и защиту от SQL-инъекций.
   - Проводить валидацию данных перед добавлением и обновлением задач в базе данных.

**Дополнительные требования:**

1. **База данных:**
   - Использовать MySQL или другую реляционную базу данных для хранения пользовательских данных и задач.

2. **Фронтенд:**
   - Создать простой и интуитивно понятный интерфейс с использованием HTML, CSS и JavaScript (по желанию).

3. **Код:**
   - Писать чистый, структурированный и комментированный код.
   - Использовать объектно-ориентированное программирование в PHP (по желанию).

4. **Документация:**
   - Предоставить краткую документацию к проекту, описывающую функционал и способ запуска приложения.

**Дедлайн:**
Веб-приложение "TaskManager" должно быть разработано и готово к проверке в течение 6 недель с момента начала работы стажера PHP программиста.

**Критерии успеха:**
1. Работоспособное веб-приложение с реализованным функционалом управления задачами.
2. Безопасность и корректность работы с базой данных.
3. Понятный и интуитивно понятный интерфейс для пользователей.
4. Документированный и комментированный код.
